# Code of Conduct

## Our Standards

**Expected behavior:**

- Be respectful and inclusive
- Accept constructive criticism gracefully
- Focus on what is best for the project

**Unacceptable behavior:**

- Personal attacks or insults
- Harassment of any kind
- Publishing others' private information

## Enforcement

Violations may result in removal from the project. Report issues to the maintainer via email or GitHub issues.

## Scope

This applies to all project spaces: issues, pull requests, discussions, and related communications.
